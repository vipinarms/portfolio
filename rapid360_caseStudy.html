Visualizing Global Threat Intelligence: A high-performance analytics dashboard for cyber security operations.

RoleLead Product Designer
Duration3 Months
EraCirca 2018
DomainCyber Security

The Challenge

In cybersecurity, data density is the enemy of speed. Rapid360° needed to synthesize millions of data points—campaigns, indicators of compromise, and global exit nodes—into a single, actionable interface that security analysts could interpret in seconds.

The Rapid360° Dark Mode Dashboard featuring real-time threat history and global exit node mapping.

Information Architecture

  • Threat History: 7-day trend analysis of IOCs.
  • Global Mapping: Geolocation of Tor Exit Nodes.
  • Campaign Metrics: Tracking active cyber operations.
  • Indicator Analysis: Granular breakdowns by Type and URL.

Core Navigation

  • Real-time Dashboard & Analytics
  • Threat Intelligence Feed
  • Global Network Monitoring
  • Campaign Management
  • Platform Configuration & Settings

Design Execution

Designed to be used in high-intensity Security Operations Centers (SOCs), I utilized a high-contrast dark theme to reduce eye strain and used color semantically to highlight critical threats (Indicators Of Compromise vs. Standard Operations).
